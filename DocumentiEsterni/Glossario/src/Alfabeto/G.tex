\section{G}

\TermineGlossario{Gamelift}
\DefinizioneGlossario{GameLift è una soluzione di hosting di server di giochi che distribuisce offerto da Amazon, gestisce e dimensiona i server cloud per giochi multigiocatore basato su sessioni.}

\TermineGlossario{Geolocalizzazione}
\DefinizioneGlossario{Processo che permette l'individuazione geografica del luogo in cui si trova un determinato oggetto attraverso l'uso di apparecchiature in grado di trasmettere segnali a un satellite.}

\TermineGlossario{Git}
\DefinizioneGlossario{È un software di controllo versione distribuito utilizzabile da interfaccia a riga di comando.}

\TermineGlossario{Git Flow}
\DefinizioneGlossario{È un’estensione di git che si basa sul modello di branching di Vincent Driessen. Git Flow descrive un modello di diramazione ben preciso costruito che rispecchia le fasi di rilascio del progetto.}

\TermineGlossario{GitHub}
\DefinizioneGlossario{È un servizio di hosting per progetti software. Il nome deriva dal fatto che è una implementazione dello strumento di controllo versione distribuito Git.}

\TermineGlossario{GitKraken}
\DefinizioneGlossario{Client GUI che permette la gestione, clonazione e sincronizzazione in locale di repository Git con funzioni trouble ticket.}

\TermineGlossario{GitLab}
\DefinizioneGlossario{È una piattaforma web open source che permette la gestione di repository Git e di funzioni trouble ticket.}

\TermineGlossario{Global Positioning System (GPS)}
\DefinizioneGlossario{È un sistema di posizionamento e navigazione satellitare. Attraverso una rete dedicata di satelliti artificiali in orbita, fornisce a un terminale mobile o ricevitore GPS informazioni sulle sue coordinate geografiche e sul suo orario in ogni posto sulla Terra dove vi sia un contatto privo di ostacoli con almeno quattro satelliti del sistema.}

\TermineGlossario{Grafana}
\DefinizioneGlossario{È un sistema di analisi e monitoraggio open source per basi di dati e funziona come un'applicazione web.}

\TermineGlossario{Gradle} 
\DefinizioneGlossario{È un sistema open source di compilazione che viene utilizzato per la automate building, i testing, le distribuizioni, ecc. Ogni progetto Android ha 
bisogno di un Gradle per generare un apk dai file .java e .xml .}

\TermineGlossario{GRPC} 
\DefinizioneGlossario{gRPC è un Framework RPC (Remote Procedure Call) a prestazioni elevate e indipendente dal linguaggio.}

\TermineGlossario{GUI (Graphical User Interface, Interfaccia grafica)}
\DefinizioneGlossario{È un tipo di interfaccia utente che consente l'interazione uomo-macchina in modo visuale utilizzando rappresentazioni grafiche piuttosto che riga di comando.}
