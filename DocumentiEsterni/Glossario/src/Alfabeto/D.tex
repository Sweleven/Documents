\section{D}
\TermineGlossario{D3.js}
\DefinizioneGlossario{Libreria JavaScript per creare visualizzazioni dinamiche ed interattive partendo da dati organizzati, visibili attraverso un comune browser. Per fare ciò si serve largamente degli standard web: SVG, HTML5, e CSS. }

\TermineGlossario{Dashboard}
\DefinizioneGlossario{È un tipo di interfaccia utente grafica che mostra informazioni generiche ed è caratterizzata dalla facilità di lettura ed immediatezza, consentendo al management di agire tempestivamente prendendo decisioni corrette.}

\TermineGlossario{Dati dell'organizzazione}
\DefinizioneGlossario{È un sottoinsieme dei parametri dell'organizzazione, nel quale vengono esclusi il perimetro di tracciamento dell'organizzazione e l'insieme di luoghi di tracciamento.}

\TermineGlossario{Design}
\DefinizioneGlossario{in italiano progettazione è un piano o una specifica per la costruzione di un oggetto o sistema o per l'implementazione di un'attività o di un processo, o (in alcuni casi ex.artigiano)il risultato di quel piano o specifica sotto forma di prototipo, prodotto o processo}

\TermineGlossario{Disciplinato}
\DefinizioneGlossario{Comportamento conforme a regole e strategie ben definite.}

\TermineGlossario{Discord}
\DefinizioneGlossario{È una piattaforma gratuita di chat vocale e testuale che fornisce server gratis per gli utenti e infrastrutture per server dedicati.}

\TermineGlossario{Dispatcher}
\DefinizioneGlossario{È un modulo del sistema operativo che ha la responsabilità di: estrazione dei messaggi in arrivo dai canali sottostanti, della loro conversione in chiamate al metodo nel codice dell'applicazione e della restituzione dei risultati al chiamante.}

\TermineGlossario{Docker}
\DefinizioneGlossario{È un progetto open source che automatizza il deployment di applicazioni all'interno di contenitori software, fornendo un'astrazione aggiuntiva grazie alla virtualizzazione a livello di sistema operativo di Linux.}

\TermineGlossario{DynamoDB}
\DefinizioneGlossario{è uno dei servizi di AWS completamente gestito ovvero, è un database non relazionale che supporta i modelli di dati,di tipo documento e di tipo chiave-valore che offre prestazioni di pochi millisecondi a qualsiasi scala}

\TermineGlossario{Draw.oi}
\DefinizioneGlossario{Piattaforma che permette di creare diagrammi direttamente dal proprio browser Web, senza dover utilizzare software o app.}
